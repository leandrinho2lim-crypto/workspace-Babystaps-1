# Minhas Anotações de Aula 📚

## 💡 Introdução à Lógica (Prof. Rafaela)
* **O que é Lógica:** O raciocínio organizado para resolver problemas.
* **Lógica de Programação:** A base para criar sequências que o computador entenda.

## 📝 Lógica na Prática (Prof. Daniela)
* **Pensamento do Programador:** Organizar problemas complexos em passos lógicos simples.
* **Por que aprender Lógica primeiro:** Para entender o "como" antes do "quê" (linguagem).
* **Variáveis vs Constantes:** 
    * *Variáveis:* Valores que podem mudar durante o programa.
    * *Constantes:* Valores que permanecem fixos.
* **Estruturas de Controle:**
    * **Sequência:** O passo a passo linear.
    * **Decisão (Condição):** O uso de "se/então" para caminhos diferentes.
    * **Repetição:** Ciclos para automatizar tarefas.
* **Pensamento Estruturado:** Estrutura básica de um algoritmo e uso de **Fluxogramas**.

## 🛠️ Ferramentas (Prof. Paloma)
* **VisualG:** Ambiente para prática de algoritmos e lógica usando português estruturado.
